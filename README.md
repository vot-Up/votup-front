## Pre-instalacao

- instale o `nvm`

- [Linux](https://gist.github.com/d2s/372b5943bce17b964a79),
- use a versao v22.13.0, usando os comando `nvm install v22.13.0` e `nvm use v22.13.0`

## Instalar depencias

- rode no terminal o comando `npm install`
