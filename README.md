# UrnaMagica

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 16.1.4.

## Pre-instalacao

- instale o `nvm`
- [Windows](https://www.freecodecamp.org/news/node-version-manager-nvm-install-guide/),
- [Linux](https://gist.github.com/d2s/372b5943bce17b964a79),
- use a versao 18.10, usando os comando `nvm install v18.10` e `nvm use v18.10`

## Instalar depencias

- rode no terminal o comando `npm install`
